# The Unselected Journal of Anjali

A personal portfolio, built as I learn web development through freeCodeCamp's
Responsive Web Design certification — framed as a journal rather than a resume.

Reading, journaling, and writing have always been how I process things.
This site is the same idea, just in HTML: one entry at a time.

## Sections

- **Hero** — who I am, right now
- **About** — why I write, why I'm learning to code
- **Currently Reading** — whatever book is shaping my thinking at the moment
- **Entries** — things I've built, what I learned from each
- **Notes** — a running build log, day by day
- **Contact** — email, GitHub, LinkedIn
- **Footer** — a quote that stuck with me that day
